# Elastic — research
Provider/key: greenhouse:elastic | company_type: product

What they do: Elastic builds the Elastic Search AI Platform — an open-source search and analytics engine (Elasticsearch) and surrounding products for search, observability, and security. Their cloud-hosted Elastic Cloud offering and on-prem Elastic Stack (Elasticsearch, Kibana, Logstash, Beats) power use cases including application search, log analytics, APM, SIEM, and vector search for AI/ML.

How they describe themselves: "Turn endless data possibilities into real results with the power of Search AI." Elastic operates on "a set of ideas — rather than a statement of corporate values — to keep us honest as we go about our business." Philosophy centers on distribution ("built distributed"), openness (open-source roots), and connection. "We care more about what you can do than where you rest your head." Serves over 50% of Fortune 500 companies.

Size / stage / funding: Public company (NYSE: ESTC). ~3,500+ employees across 40+ countries, representing 100+ nationalities. Founded 2012, Mountain View CA. Revenue ~$1.5B ARR (FY2025). Enterprise-scale mature public company.

Locations / HQ: HQ in Mountain View, CA (San Francisco Bay Area). Distributed globally across 40+ countries. No single dominant office — distributed by design and intention. Has a dedicated Canada openings page at jobs.elastic.co.

Remote policy: Distributed by design. "Built distributed and cares more about what colleagues do rather than where they rest their heads." Offers remote, hybrid, and in-person roles depending on the specific team and location. Remote-first culture is a founding principle, not a policy reaction.

Remote-Canada eligibility: Yes — verified. Elastic has a dedicated Canada jobs page (jobs.elastic.co/jobs/country/canada) listing remote roles. Salary ranges for Canada-based remote roles include $146k–$232k. Remote Rocketship lists "Elastic AI Engineer - Canada Remote." Multiple sources confirm active Canada hiring for engineering roles. Elastic is known as one of the most distributed-friendly engineering employers.

Engineering & tech: Distributed systems at scale (Elasticsearch is a large-scale Java distributed search engine). Roles in the batch focus on: networking for platform infrastructure, search scaling, vector search, Java distributed systems. Primary language: Java (Elasticsearch core). Also Go, Python. Tech: Lucene, Raft consensus, distributed indexing, shard management, vector indexing (HNSW), Kubernetes, cloud-native deployment (AWS, GCP, Azure via Elastic Cloud).

Notable / other: Elasticsearch is one of the most widely deployed open-source software projects. Elastic has strong engineering credentials — Principal and Senior engineers work on genuinely hard distributed systems problems. Transparent engineering blog, active open-source community. ESRE (Elasticsearch Relevance Engine) and vector search are current growth areas. Acquired multiple companies (Endgame, Cmd.io, etc.).

Open relevant roles (sample): Principal Software Engineer (Networking) Platform, Principal Software Engineer Search Scaling Elasticsearch, Principal Software Engineer Vector Search Elasticsearch, Senior Java Developer Distributed Systems Elasticsearch, Senior Java Engineer Distributed Systems Elasticsearch, Senior Software Engineer Docs Engineering, Senior Software Engineer Vector Search Elasticsearch
