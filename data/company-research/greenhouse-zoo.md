# Zoo — research
Provider/key: greenhouse:zoo | company_type: product

What they do: Zoo (formerly KittyCAD) builds next-generation CAD (computer-aided design) software and hardware design infrastructure. Their product suite includes: Zoo Design Studio (a full-stack AI-native CAD application for hardware engineers), Zookeeper (a conversational CAD agent for reasoning-driven design), and a public Design API for developers to build hardware design tooling. Their mission is to modernize CAD — a space dominated by decades-old legacy software (SolidWorks, AutoCAD, CATIA).

How they describe themselves: "Next-Gen CAD Built for the Modern Era" and "leading a revolution in hardware engineering." Focus on combining traditional CAD workflows with AI-assisted and conversational interfaces. Targets hardware engineers who currently suffer from outdated tooling.

Size / stage / funding: Small startup — ~38 employees as of 2025. Total funding: ~$10.12M (multiple Seed rounds). Pre-Series A, early-stage. Founded 2021 in Inglewood/Los Angeles, California. Co-founders: Jessie Frazelle (ex-Microsoft, Docker, Google), Hannah Bollar, Jordan Noone, Jenna Bryant.

Locations / HQ: HQ in Inglewood/Los Angeles, California. Describes having a "globally distributed team" but careers page lists all positions as "Los Angeles, California, United States; Remote." No Canada-specific mention in any job listings.

Remote policy: Remote options available for most software engineering roles (CAD Graphics Engine, Cloud Infrastructure, Language Development, Systems, Frontend all listed as remote-eligible). Some physical roles (Lead Machinist, Manufacturing Engineer) are LA-only. However, "remote" appears to mean remote from LA headquarters, with no explicit geographic qualification by country.

Remote-Canada eligibility: Unclear / unconfirmed. The careers page lists roles as "Los Angeles, California, United States; Remote" — the "United States" qualifier is present in the location string. No Canada-remote listings found anywhere. No Canada-specific mention on website or job boards. With only ~38 employees, Canada hiring infrastructure (payroll, benefits) is uncertain.

Engineering & tech: Rust (primary systems/backend language, consistent with founders' backgrounds). WebAssembly (WASM). 3D graphics engine, geometry kernel (custom/modern), CAD file format parsing. Cloud infrastructure (Kubernetes, cloud-native). Language development (KCL — KittyCAD Language, a domain-specific language for parametric design). OpenAPI/REST APIs. Low-level systems programming.

Notable / other: Jessie Frazelle is a prominent open-source/systems figure (authored many container security tools). Company is open-source oriented. Competing against billion-dollar legacy CAD vendors (Dassault, Autodesk, PTC). Very small team relative to ambition. Interesting technical problem space (hardware design infrastructure) but early-stage and niche.

Open relevant roles (sample): CAD Graphics Engine Software Engineer, Senior Software Engineer Cloud Infrastructure, Senior Software Engineer Language Development, Senior Systems Software Engineer
