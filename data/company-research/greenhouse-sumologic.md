# Sumo Logic — research
Provider/key: greenhouse:sumologic | company_type: product

What they do: Sumo Logic is a cloud-native SIEM (Security Information and Event Management), log management, and observability platform. It ingests, indexes, and analyzes machine-generated log data at scale for security operations, compliance, infrastructure monitoring, and DevSecOps workflows. Products include Cloud SIEM, log analytics, threat intelligence, AI-powered anomaly detection (now branded "Dojo AI" multi-agent platform), and integrations with AWS, Azure, and GCP. Core ingest and data query engines are large-scale distributed systems.

How they describe themselves: "Cloud-native machine data analytics platform that delivers real-time continuous intelligence." Values: collaboration, ownership, customer focus, and continuous improvement. Tone is professional/enterprise. Post-acquisition, emphasis on AI/agentic security operations. Culture described as collaborative and growth-oriented by employees.

Size / stage / funding: ~838–983 employees (2025/26 estimates vary). Previously public (NASDAQ: SUMO); taken private by Francisco Partners in May 2023 for $1.7B ($12.05/share). Now a PE-owned private company. Has raised $346M+ over 7 prior VC rounds. Backers included Accel, Battery, Greylock, Sequoia, Tiger Global, Sutter Hill.

Locations / HQ: Headquarters in Redwood City, CA. Offices in Denver, CO. Engineering centers in India (Noida, Bangalore) and Costa Rica (San José). US remote roles available.

Remote policy: Hybrid with significant remote presence for US-based roles. US remote roles confirmed on job board. Engineering & tech: Java, Go, Python. Distributed data ingest pipeline (Core Ingest team), distributed query engine (Data Query team), OpenTelemetry collector (Go), ML/data collection systems. Very large-scale data processing — billions of log lines per day. Infrastructure on AWS, multi-region. The Core Ingest and Data Query roles are distributed-systems engineering at scale.
