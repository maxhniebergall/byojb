# Docker — research
Provider/key: ashby:docker | company_type: product

What they do: Docker builds and maintains the Docker containerization platform — the industry-standard tooling for building, packaging, and running containerized applications. Products include Docker Desktop, Docker Hub, Docker Scout (security), and Docker Build Cloud. Used by millions of developers daily. They pivoted post-2019 (after selling the enterprise division to Mirantis) to focus on developer tooling and experience.

How they describe themselves: "Docker drives modern software development by making it easy to adopt container technology to radically boost productivity, security, testing, and collaboration at every step of the developer experience." Five core virtues: (1) Developer Obsession — customer-centric; (2) Humble Confidence — ego-free, kind, learning-focused; (3) Bias for Considered Action — data-driven, fast-moving; (4) Open Collaboration — transparency, breaking silos; (5) Outcome Driven — goal-focused with critical thinking. Quarterly hackathons. "Growing at a record pace."

Size / stage / funding: Private. "Hundreds of people building software." Offices in Paris and Seattle (optional; remote-first). Exact headcount not publicly disclosed. Late-stage private company.

Locations / HQ: No physical HQ in traditional sense (fully remote-first). Optional office hubs in Paris and Seattle. Team spans Europe, APAC, and the Americas.

Remote policy: Fully remote-first. "Docker is a remote first company with employees across Europe, APAC and the Americas." No physical HQ requirement. Day-to-day work is fully remote. Async-friendly. Optional office hub access in Paris and Seattle. Remote work stipend provided.

Remote-Canada eligibility: Yes — confirmed. Multiple job listings explicitly include "US/Canada Eastern or European CET/Western Regions" (e.g., Staff Backend Engineer, Secure Policy; Senior Backend Engineer, Secure Policy). Docker is listed on remote job boards as hiring in Canada. Senior Frontend Software Engineer explicitly listed as "Canada Remote" on RemoteRocketship.

Engineering & tech: Container technology, Docker Desktop, Hub, Agents (AI-integrated containers), AI Tools and Security, Data Engineering. Stack spans Go, Rust, TypeScript/JavaScript, Docker's own container runtime layers. Separate teams for Desktop, Hub, Agents, Infrastructure, AI Tools & Security.

Notable / other: Docker is iconic in the developer tools space — strong brand recognition. Post-2019 focus on developer experience side of containers vs. enterprise orchestration. Recent "Docker Agents" product push suggests AI integration. Some roles explicitly location-restricted to East Coast US or London.

Open relevant roles (sample): Staff Software Engineer, Data Engineering; Senior Principal Software Engineer, Infrastructure; Senior Software Engineer, AI Tools and Security; Principal Software Engineer, AI Tools and Security (US West Coast)
