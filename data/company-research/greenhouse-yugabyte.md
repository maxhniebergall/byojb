# YugabyteDB — research
Provider/key: greenhouse:yugabyte | company_type: product

What they do: YugabyteDB is a distributed SQL database available as open-source software (Apache 2.0) and as a managed cloud service (YugabyteDB Astra). Designed for cloud-native transactional applications requiring global scale and high availability. Supports PostgreSQL-compatible API (YSQL) and Cassandra-compatible API (YCQL). Founded 2016 by former Facebook engineers (Karthik Ranganathan, Mikhail Bautin, others).

How they describe themselves: Three core values: Trust ("We build trust through humility, integrity, and transparency"), Relationships (prioritizing connections with colleagues, customers, community), Outcomes ("committed to technical excellence, ownership, and making a positive impact"). Mission: providing "a data platform that's infinite, everlasting, and adds value" (name blends Sanskrit "yuga" for era with "byte"). Described as a "people-first culture" with transparency and distributed collaboration. Committed to open-source principles since 2019 when all features were made Apache 2.0.

Size / stage / funding: Series C (~2021). $291M total raised. Investors: Lightspeed Venture Partners, Sapphire Ventures, Dell Tech Capital, 8VC, Alkeon Capital, Meritech Capital, Wells Fargo (strategic), Wipro Ventures. Valuation: $1.3B (2021 Series C). ~423 employees as of early 2026, down from peak ~526 in 2023 (some contraction). No new major funding rounds since 2021.

Locations / HQ: Sunnyvale CA (HQ). Singapore, Bangalore India, Pune India, Tokyo Japan. No listed Canadian office. One Support Engineer role found explicitly open to British Columbia and Ontario, Canada (CAD $90K–125K salary range).

Remote policy: Multiple models: fully remote (independent, no office requirement), flexible (hybrid 1-2 days/week), in-office (3-5 days). Company describes "flexible, distributed work culture." Glassdoor reviews highlight remote work and WLB as positives.

Remote-Canada eligibility: Likely for some roles — BC and Ontario explicitly listed in at least one job posting. However, the Platform Software Engineer (Control Plane) role was listed "Remote, USA" only. Eligibility appears role-dependent; engineering roles may be US-only while support/other roles are open to Canada. Verify per listing.

Engineering & tech: Distributed SQL database internals, PostgreSQL compatibility layer, Cassandra-compatible API, distributed transactions, distributed storage engine, control plane software (Kubernetes, cloud orchestration), cloud platform engineering. Stack: Java, Go, C++, Python, Kubernetes, Docker. 1M+ TPS benchmark, 10K+ GitHub stars, 100+ country deployments.

Notable / other: Open-source first philosophy (Apache 2.0 all features since 2019). Competing primarily with CockroachDB, Amazon Aurora, Google Spanner. Community of 11,000+ members. Company has experienced some employee count contraction since 2023 peak (~526 to ~423). Solid but not hyper-growing; stable, well-established technical product in the distributed database space.
