Provider/key: greenhouse:dbtlabsinc | company_type: product
What they do: dbt Labs develops dbt (data build tool), an open-source command-line tool and managed cloud service (dbt Cloud) that enables data analysts and engineers to transform data in their warehouses using software engineering best practices (like version control, testing, and documentation).
How they describe themselves: Dedicated to helping data teams "create and disseminate organizational knowledge." They foster a "people-first" culture that values transparency ("transparency always wins"), inclusivity, humility, and contributing to the open knowledge ecosystem. They operate with a strong "async-first" written culture.
Size / stage / funding: Private venture-backed startup, established growth stage (unicorn status), ~300-500 employees.
Locations / HQ: Philadelphia, Pennsylvania (HQ), but fully distributed.
Remote policy: Remote-first, async-first. Focuses heavily on documentation and written communication.
Remote-Canada eligibility: Verified yes. They actively hire and support remote software and analytics engineering roles across Canada.
Engineering & tech: Heavy focus on SQL compiler translation, database execution engines, data storage, and SaaS backend platforms. Tech stack includes Python (for dbt-core development), Go, TypeScript/React, AWS, GCP, and cloud data warehouses (Snowflake, BigQuery, Databricks).
Notable / other: Hosts a prominent tech and data engineering blog (getdbt.com/blog) discussing analytics engineering paradigms and AI integration (including AI/agentic data workflows).
Open relevant roles (sample): Senior Backend Engineer, dbt Cloud; Software Engineer, Semantic Layer; Data Infrastructure Engineer.
